# 🏦 Banking Management System

A Banking Management System developed using Java Servlets, JSP, and MySQL that allows users to manage bank accounts, perform transactions, and view account details securely. The system supports core banking functionalities such as deposits, withdrawals, fund transfers, and transaction history.

---

## 🚀 Features

### 👤 User Module
- User registration and login (secure authentication)
- Create and manage bank accounts
- Deposit and withdraw money
- Transfer funds between accounts
- View account balance
- Transaction history tracking

### 🛠️ Admin Module *(Optional if implemented)*
- Manage user accounts
- Monitor transactions
- View system activity

---

## 🧑‍💻 Technologies Used

- **Frontend:** HTML, CSS, JSP  
- **Backend:** Java Servlets  
- **Database:** MySQL  
- **Connectivity:** JDBC  
- **Server:** Apache Tomcat

## ⚙️ Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/banking-management-system.git
   ```

2. Import project into IDE:
   - Eclipse / IntelliJ as Dynamic Web Project

3. Configure Apache Tomcat:
   - Add project to server

4. Setup MySQL Database:
   - Create database (e.g., `bank_db`)
   - Import SQL file
   - Update DB credentials in connection file

5. Add JDBC Driver:
   - Place MySQL Connector JAR in `/lib`

6. Run the application:
   - Start Tomcat server
   - Open browser:
     ```
     http://localhost:8080/Banking-Management-System
     ```
